---
title: "whileを評価する"
---

はい、繰り返し（`while`）を作ってませんでしたね！

静的スコープを大モノ扱いしたので while が小モノみたいな扱いになってしまってますが順次・選択に加えて繰り返しがあればすべてのアルゴリズムが書けるということになっているのでやはり大事なものです[^recursive-school]。

[^recursive-school]: if と関数があれば再帰で繰り返しが書けるから繰り返しなんかいらないよ、という宗派もあります。

`while` は制御構造なので `seq` や `if` と同じように書いていきます。

```diff py
 class Evaluator:
     def eval(self, expr, env):
         match expr:
             ...
             case ("if", [cond_expr, then_expr, else_expr]):
                 return self._if(cond_expr, then_expr, else_expr, env)
+            case ("while", [cond_expr, body_expr]):
+                return self._while(cond_expr, body_expr, env)
             ...
```

`("while", [cond_expr, body_expr])` という形の式だったら、`_while()` を呼びます。

```diff py
     def _if(self, cond_expr, then_expr, else_expr, env):
         ...

+    def _while(self, cond_expr, body_expr, env):
+        val = None
+        while self.eval(cond_expr, env): val = self.eval(body_expr, env)
+        return val
```

`_while()` では、`cond_expr` を評価して真である間 `body_expr` を繰り返し評価します。`val` に結果を覚えておき、繰り返しが終わったらその値を返します。最後に評価した式の値を返すことになります。最初から `cond_expr` が偽であれば `None` が返ります。

実行例行きましょう。ちょっと長い式になりますが、そろそろわりと普通に読めてきたのではないでしょうか？`less` は `<` のことです。

```py
    print("While:")

    print(toil.eval(("seq", [
        ("define", ["i", 1]),
        ("while", [
            ("less", ["i", 4]),
            ("seq", [
                ("print", ["i"]),
                ("assign", ["i", ("add", ["i", 1])])
            ])
        ])
    ])))
    # -> 1\n2\n3\n4
```

`i` が 4 より小さい間 `i` を `print` します。`1`、`2`、`3` は print の実行結果で、`4` は while 文の返した値、つまり `("assign", ["i", ("add", ["i", 1])])` を最後に実行したときの値です。

```py
    print(toil.eval(("seq", [
        ("define", ["i", 1]), ("define", ["sum", 0]),
        ("while", [
            ("less", ["i", 4]),
            ("seq", [
                ("assign", ["sum", ("add", ["sum", "i"])]),
                ("assign", ["i", ("add", ["i", 1])])
            ])
        ]),
        "sum"
    ])))
    # -> 6
```

ちょっとだけ複雑な例。1 から 3 までの数の合計を計算しています。合計を返すために最後に `sum` だけの式を書いています。`return sum` のようなものです。

```py
    print(toil.eval(("while", [False, ("div", [1, 0])])))
    # -> None
```

`while` の本体が一度も実行されなかったときは `None` を返します。while の本体は `("div", [1, 0])` で、もし実行されてしまったら `ZeroDivisionError` になります。今後も、実行されてはいけない箇所にときどき書くことがあります。

```py
    print(toil.eval(("seq", [
        ("define", ["i", 0]),
        ("while", [
            ("less", ["i", 2]),
            ("seq", [
                ("define", ["j", 0]),
                ("while", [
                    ("less", ["j", 3]),
                    ("seq", [
                        ("print", ["i", "j"]),
                        ("assign", ["j", ("add", ["j", 1])])
                    ])
                ]),
                ("assign", ["i", ("add", ["i", 1])])
            ])
        ])
    ])))
    # -> 0 0\n0 1\n0 2\n1 0\n1 1\n1 2\n2
```

`while` が入れ子になっても大丈夫。

おつかれさまです！これで、第 1 章で実装予定の機能をすべて実装しました。
