変数を導入します。変数を導入するからには変数を保存しておく場所が必要ですが、第 1 章で作った `Environment` をそのまま使います。ということさえ決めてしまえばあとは特に難しいことはありません[^memory-management]。

[^memory-management]: 一般的な言語ではここでメモリ上にどのように値を置くか考えるところなんですけれども Toil はそこには踏み込まないことにします。

仮想マシンに変数を定義する `def` と変数の値を取得する `get` のふたつの命令を追加します。

```diff py
 class VM:
-    def __init__(self, code):
+    def __init__(self, code, env):
         self._code = code
+        self._env = env
         self._ip = 0
         self._stack = []
```

`_env` に環境を覚えるようにします。

```diff py 
     def execute(self):
         while (inst := self._code[self._ip]) != ("halt",):
             self._ip += 1
             match inst:
                 case ("const", val): self._stack.append(val)
                 case ("pop",): self._stack.pop()
+                case ("def", name): self._env.define(name, self._stack[-1])
+                case ("get", name): self._stack.append(self._env.val(name))
                 case ("print",):
                     print(self._stack.pop()); self._stack.append(None)
                 ...
             ...
```

`case ("def", name):` ではスタックの一番上にある値を `name` という名前で環境に記録します。
入力をスタックから持ってきて結果をスタックに置く、というのをそのまま書くとこうなりますが、

```py
                case ("def", name): 
                    val = self._stack.pop()
                    self._env.define(name, val)
                    self._stack.append(val)
```

`pop()` で持ってきたものをそのまま `append()` するだけで実際にはスタックにはなんの変化もなく、ただスタック最上位の値が欲しいだけなので `self._stack[-1]` で値を取得しています。`a := 2` という式は `2` という値を持つことを思い出してください。

`case ("get", name):` のほうは素直に `name` という名前の変数の値をスタックに積むだけです。これで変数の値を取得したことになります。

コンパイラの方にも変数の定義と参照を追加します。

```diff py
class Compiler:
     ...
 
     def _expression(self, expr):
         match expr:
             case None | bool() | int(): self._emit("const", expr)
+            case str(name): self._emit("get", name)
+            case ("define", [name, expr]):
+                self._expression(expr)
+                self._emit("def", name)
             case ("seq", exprs): self._seq(exprs)
                 ...
             ...

    def _seq(self, exprs):
         ...
```

`expr` が文字列ならば変数名なので、`get` 命令を出力して変数の値がスタックに積まれるようにします。
`("define", [name, expr])` であれば `self._expression(expr)` で値を計算するコードを出力し、さらに `def` で計算結果に名前を付けて変数を定義するようにします。

`Interpreter` クラスでは `execute()` で仮想マシンに `_env` を渡すように修正します。

```diff py
class Interpreter:
     ...

     def execute(self, code):
-        return VM(code).execute()
+        return VM(code, self._env).execute()

    ...
```

実行例です。

```py
    print("Variable and definition:")

    print(toil.ast(r""" a := 2 """))
    # -> ('define', ['a', 2])
    print_code(toil.code(r""" a := 2 """))
    # ->   0: ('const', 2)
    # ->   1: ('def', 'a')
    # ->   2: ('halt',)
    print(toil.run(r""" a := 2 """)) # -> 2

    print(toil.ast(r""" a """))
    # -> a
    print_code(toil.code(r""" a """))
    # ->   0: ('get', 'a')
    # ->   1: ('halt',)
    print(toil.run(r""" a """)) # -> 2

    print(toil.ast(r""" a := b := 2 == 2 """))
    # -> ('define', ['a', ('define', ['b', ('equal', [2, 2])])])
    print_code(toil.code(r""" a := b := 2 == 2  """))
    # ->   0: ('const', 2)
    # ->   1: ('const', 2)
    # ->   2: ('equal',)
    # ->   3: ('def', 'b')
    # ->   4: ('def', 'a')
    # ->   5: ('halt',)
    print(toil.run(r""" a := b := 2 == 2  """)) # -> True
    print(toil.run(r""" a """)) # -> True
    print(toil.run(r""" b """)) # -> True

    print(toil.run(r""" a := 2; b := 3; a * b """)) # -> 6

    # toil.run(r""" c """) # -> Undefined variable
```

それほど難しいところはないと思いますが、 `a := b := 2 == 2` で `2 == 2` の結果を `b` の定義と `a` の定義で使いまわしているところはコードと見比べておいてください。
