title: "3.9 while"
---

`while i < 3 do i = i + 1 end` は以下のようにコンパイルされます。`i` には `1` が入っているところから始まっていると思ってください。

```py
 0: ('const', None)
 1: ('get', 'i')
 2: ('const', 3)
 3: ('less',)
 4: ('jump_if_false', 11)
 5: ('pop',)
 6: ('get', 'i')
 7: ('const', 1)
 8: ('add',)
 9: ('set', 'i')
10: ('jump', 1)
11: ('halt',)
```

解説します。

```py
 0: ('const', None)
```

最初にスタックに `None` を積んでおきます。これは、ループを一度も実行しなかったときの `while` の値になります。

```py
 1: ('get', 'i')
 2: ('const', 3)
 3: ('less',)
```

`i < 3` の値をスタックに積みます。

```py
 4: ('jump_if_false', 11)
```

偽だったらループを抜けます。`11` という飛び先アドレスはこの時点では判明していないので、バックパッチを使います。

ここからループの中に入ります。

```py
 5: ('pop',)
```

前回のループの結果がスタックに残っているので捨てます。最初に積んだ `None` もたいていはここで捨てられます。

```py
 6: ('get', 'i')
 7: ('const', 1)
 8: ('add',)
 9: ('set', 'i')
```

`i = i + 1` を実行し、値（ここでは `i + 1`）をスタックに積みます。

ここまでがループの中身です。

```py
10: ('jump', 1)
```

`while` の先頭に戻ります。先頭の番地はわかっているのでバックパッチは使いません。先頭の番地を覚えておく必要はあります。


```py
11: ('halt',)
```

ループを抜けました。バックパッチで `jump_if_false` のオペランドを `11` に書き換えます。

スタックにはループ内で最後に実行した `i = i + 1` の値が残っており、これが `while` の値となります。この式では `3` です。

`while` も `jump` と `jump_if_false` で書けました。なので `VM` には修正はありません。

`Compiler` の修正です。

```diff py
class Compiler:
     ...
 
     def _expression(self, expr):
         match expr:
             ...
             case ("if", [cond_expr, then_expr, else_expr]):
                 self._if(cond_expr, then_expr, else_expr)
+            case ("while", [cond_expr, body_expr]):
+                self._while(cond_expr, body_expr)
             case (op, [expr]):
                 ...
             ...

     ...

     def _if(self, cond_expr, then_expr, else_expr):
         ...
 
+    def _while(self, cond_expr, body_expr):
+        self._emit("const", None)
+        loop_jump = self._current_addr()
+        self._expression(cond_expr)
+        cond_jump = self._current_addr()
+        self._emit("jump_if_false", None)
+        self._emit("pop")
+        self._expression(body_expr)
+        self._emit("jump", loop_jump)
+        self._set_operand(cond_jump, self._current_addr())

     def _set_operand(self, ip, operand):
         ...
```

中間コードの解説と見比べながら読むと、そのままコードになっているのがわかると思います。

`while` の先頭（条件の式を実行するところ）で `loop_jump` に番地を覚えておき、ループの中身を終えたところで `self._emit("jump", loop_jump)` でそこに戻っています。わかっている番地なのでそのまま `loop_jump` 番地に `jump` すると書けばよく、バックパッチは要りません。

実行例です。

```py
    print("While:")

    print(toil.ast(r""" while i < 3 do i = i + 1 end """))
    # -> ('while', [('less', ['i', 3]), ('assign', ['i', ('add', ['i', 1])])])
    print_code(toil.code(r""" while i < 3 do i = i + 1 end """))
    # ->   0: ('const', None)
    # ->   1: ('get', 'i')
    # ->   2: ('const', 3)
    # ->   3: ('less',)
    # ->   4: ('jump_if_false', 11)
    # ->   5: ('pop',)
    # ->   6: ('get', 'i')
    # ->   7: ('const', 1)
    # ->   8: ('add',)
    # ->   9: ('set', 'i')
    # ->  10: ('jump', 1)
    # ->  11: ('halt',)
    print(toil.run(r""" i := 1; while i < 3 do i = i + 1 end """)) # -> 3
```

説明したとおりです。

```py
    print(toil.run(r"""
        sum := 0;
        i := 1; while i < 4 do sum = sum + i; i = i + 1 end;
        sum
    """))  # -> 6
```

少しだけ複雑に。

```py
    toil.run(r"""
        i := 1; while i < 3 do
            j := 1; while j < 3 do print(i); print(j); j = j + 1 end;
            i = i + 1
        end
    """)  # -> 1\n1\n1\n2\n2\n1\n2\n2
```

入れ子にしても大丈夫。内側の `while` も `<式>` ですので `_expression()` で普通に処理できます。

```py
    print(toil.run(r""" while False do 1/0 end """)) # -> None
```

ループ内を一度も通らない場合も想定通り動きました。

これで制御構造ができて、あとは関数です。まずは組み込み関数から。
